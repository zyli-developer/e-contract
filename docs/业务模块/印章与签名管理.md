# 模块：印章与签名管理

## 一、模块概述

印章与签名是电子合同签署的核心凭证。平台支持个人签名（手写/模板）和企业印章（公章/合同章/财务章/法人章），并提供企业印章的授权管理机制，确保印章使用合规可控。

## 二、印章类型

### 2.1 类型定义

| 类型码 | 名称 | 归属 | 说明 |
|--------|------|------|------|
| **企业印章** | | | |
| 1 | 企业公章 | 企业 | 代表企业对外的最高效力印章 |
| 2 | 财务专用章 | 企业 | 用于财务相关合同 |
| 3 | 合同专用章 | 企业 | 日常合同签署用章 |
| 4 | 法定代表人章 | 企业 | 法人代表个人章 |
| **个人印章** | | | |
| 11 | 个人签名 | 个人 | 手写签名样式 |
| 12 | 个人印章 | 个人 | 印章样式的个人签名 |

### 2.2 印章状态

| 状态码 | 名称 | 说明 |
|--------|------|------|
| 1 | 正常 | 可正常使用 |
| 2 | 已吊销 | 不可使用，已停用 |

## 三、印章创建

### 3.1 创建方式

| 方式 | 说明 |
|------|------|
| 模板生成 | 从系统预设的印章模板中选择，自动填入名称生成 |
| 手写绘制 | 在画布上手写签名/绘制印章 |
| 图片上传 | 上传已有的印章/签名图片 |

### 3.2 创建流程

```
用户进入印章管理 → 点击"创建印章"
     │
     ├── 选择印章类型（个人签名/个人印章/企业章）
     │
     ├── 选择创建方式：
     │     ├── [模板] 选择模板 → 输入名称 → 预览 → 确认
     │     ├── [手写] 进入画布 → 手写签名 → 预览 → 确认
     │     └── [上传] 选择图片 → 裁剪调整 → 预览 → 确认
     │
     └── POST /seal/seal-info/create → 印章创建成功
```

### 3.3 印章模板

系统提供预制印章模板：
```
GET /api/v1/seal/seal-template/page → 获取可用模板列表
POST /api/v1/seal/seal-template/use → 使用模板创建印章
```

### 3.4 默认印章

每种类型可设置一个默认印章，签署时自动使用：
```
PUT /api/v1/seal/seal-info/set-default → 设为默认印章
```

## 四、企业印章授权

### 4.1 授权机制

企业印章需要通过授权才能被成员使用：

```
企业管理员 → 印章授权管理
     │
     ├── 查看印章的授权成员列表
     ├── 添加授权：选择成员 → 授权使用指定印章
     └── 撤销授权：移除成员的印章使用权
```

### 4.2 授权规则

| 规则 | 说明 |
|------|------|
| 授权主体 | 企业 Owner 或 Admin 可管理授权 |
| 授权粒度 | 按印章维度，一枚印章可授权给多个成员 |
| 即时生效 | 授权/撤销实时生效 |
| 签署校验 | 签署时校验当前用户是否有该印章的使用权 |

### 4.3 授权 API

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/seal/grant/create` | POST | 授权成员使用印章 |
| `/api/v1/seal/grant/delete` | DELETE | 撤销成员的印章授权 |
| `/api/v1/seal/grant/page` | GET | 查询印章的授权列表 |

## 五、印章管理操作

| 操作 | 个人 | 企业 Owner/Admin | 企业 Member |
|------|------|-------------------|-------------|
| 查看印章列表 | 自己的印章 | 企业所有印章 | 被授权的印章 |
| 创建印章 | 可以 | 可以 | 不可以 |
| 编辑印章 | 可以 | 可以 | 不可以 |
| 删除印章 | 可以 | 可以 | 不可以 |
| 设为默认 | 可以 | 可以 | 不可以 |
| 授权管理 | 不适用 | 可以 | 不可以 |
| 使用印章签署 | 自己的印章 | 所有企业印章 | 被授权的印章 |
