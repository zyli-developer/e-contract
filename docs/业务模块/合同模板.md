# 模块：合同模板

## 一、模块概述

合同模板模块提供标准化的合同模板市场，用户可以浏览、搜索、预览模板，并基于模板快速创建合同。模板包含预定义的变量字段和签署方配置，降低合同创建门槛。

## 二、模板分类

| 分类 | 英文标识 | 典型模板 |
|------|---------|---------|
| 全部 | all | 所有模板 |
| 借贷合同 | loan | 借款协议、担保合同 |
| 租赁合同 | lease | 房屋租赁、设备租赁 |
| 劳动合同 | labor | 劳动合同、保密协议、竞业限制 |
| 采购合同 | purchase | 采购合同、供货协议 |
| 销售合同 | sales | 销售合同、代理协议 |
| 其他 | other | 服务合同、合作协议等 |

## 三、模板内容结构

```
模板包含：
  ├── 基本信息：
  │     ├── 模板名称
  │     ├── 模板分类
  │     ├── 使用次数
  │     └── 模板描述
  │
  ├── 模板变量（可填字段）：
  │     ├── text    - 文本输入
  │     ├── number  - 数字输入
  │     ├── date    - 日期选择
  │     ├── select  - 下拉选择
  │     └── money   - 金额输入
  │
  └── 签署方配置：
        ├── 签署方数量
        ├── 各签署方角色标识
        └── 签署位置映射
```

## 四、使用流程

```
浏览模板市场 → 选择分类/搜索关键词
     │
     ├── 查看模板详情
     │     ├── 模板说明
     │     ├── H5 预览（WebView）
     │     └── 使用次数、评价
     │
     └── 点击"使用此模板"
              │
              ├── 填写模板变量（合同双方名称、金额、日期等）
              ├── 指定签署方信息
              └── 创建合同签署任务
```

## 五、模板数据来源

| 来源 | 说明 |
|------|------|
| 平台预置模板 | 系统内置的标准化合同模板 |
| 热门模板 | 使用次数最多的模板 |
| 常用模板 | 当前用户最近使用过的模板 |

## 六、相关页面

## 七、相关 API 接口

| 接口 | 方法 | 说明 |
|------|------|------|

| `/api/v1/seal/seal-template/categories` | GET | 模板分类列表 |
| `/api/v1/seal/seal-template/frequently-used` | GET | 常用模板 |

| `/api/v1/seal/seal-template/{id}/usage` | POST | 记录模板使用 |
