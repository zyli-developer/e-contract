# 模块：企业管理

## 一、模块概述

企业管理模块支持用户创建企业、完成企业认证、管理企业成员，并在个人身份与企业身份之间切换。企业身份下可使用企业印章签署合同，具有更高的法律效力。

## 二、企业生命周期

### 2.1 企业状态

| 状态码 | 名称 | 说明 |
|--------|------|------|
| 0 | 未认证 | 已创建但未提交认证 |
| 1 | 认证中 | 已提交，等待审核 |
| 2 | 已认证 | 审核通过，功能完全可用 |
| 3 | 认证失败 | 审核不通过，需重新提交 |

### 2.2 创建企业流程

```
用户进入企业列表 → 点击"添加企业"
     │
     ├── 填写企业信息：
     │     ├── 企业名称（必填）
     │     ├── 统一社会信用代码（必填）
     │     ├── 营业执照照片（必填）
     │     ├── 法定代表人姓名
     │     ├── 法定代表人身份证号
     │     ├── 法人身份证正反面
     │     └── 联系人信息
     │
     └── POST /member/enterprise/create → 创建成功（未认证状态）
              │
              └── POST /member/verify/submit-enterprise → 提交认证
```

## 三、企业角色

| 角色码 | 名称 | 权限 |
|--------|------|------|
| 1 | Owner（所有者） | 全部权限：管理企业信息、成员、印章、授权 |
| 2 | Admin（管理员） | 管理成员、印章授权、使用所有企业印章 |
| 3 | Member（成员） | 使用被授权的印章签署合同 |

## 四、成员管理

### 4.1 添加成员

```
管理员进入成员管理 → 点击"邀请成员"
     │
     ├── 输入被邀请人手机号
     ├── 设置角色（Admin / Member）
     │
     └── POST /member/enterprise-user/create → 添加成功
```

### 4.2 成员操作

| 操作 | Owner | Admin | Member |
|------|-------|-------|--------|
| 查看成员列表 | 可以 | 可以 | 可以 |
| 邀请成员 | 可以 | 可以 | 不可以 |
| 修改成员角色 | 可以 | 仅可管理 Member | 不可以 |
| 移除成员 | 可以 | 仅可移除 Member | 不可以 |
| 转让 Owner | 可以 | 不可以 | 不可以 |

### 4.3 成员信息

```
成员列表显示：
  ├── 姓名 / 手机号
  ├── 角色（Owner / Admin / Member）
  ├── 加入时间
  └── 操作按钮（编辑角色、移除）
```

## 五、身份切换

### 5.1 个人 ↔ 企业身份

用户可在个人身份和企业身份之间自由切换：

```
首页 / 个人中心 → 身份切换下拉
     │
     ├── 个人身份：使用个人签名，合同归属个人
     │
     └── 企业身份（选择具体企业）：
              ├── 使用企业印章
              ├── 合同归属企业
              └── POST /member/enterprise/switch → 切换活跃企业
```

### 5.2 身份影响范围

| 功能 | 个人身份 | 企业身份 |
|------|---------|---------|
| 合同列表 | 显示个人合同 | 显示企业合同 |
| 可用印章 | 个人签名/印章 | 企业印章（需授权） |
| 配额余额 | 个人配额 | 企业配额 |
| 合同签署 | 以个人名义签署 | 以企业名义签署 |

### 5.3 身份存储

```javascript
// localStorage 中存储当前身份
currentIdentity: {
    type: 'personal' | 'enterprise',
    id: string,           // 企业 ID（企业身份时）
    name: string,         // 企业名称
    verified: boolean,    // 是否已认证
    role: number          // 当前用户在企业中的角色
}
```

| 页面 | 路由 | 功能 |

## 七、相关 API 接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/v1/member/enterprise/create` | POST | 创建企业 |
| `/api/v1/member/enterprise/get` | GET | 企业详情 |
| `/api/v1/member/enterprise/list` | GET | 企业列表 |
| `/api/v1/member/enterprise/delete` | DELETE | 删除企业 |
| `/api/v1/member/enterprise/switch` | POST | 切换活跃企业 |
| `/api/v1/member/verify/submit-enterprise` | POST | 提交企业认证 |
| `/api/v1/member/enterprise-user/create` | POST | 添加成员 |
| `/api/v1/member/enterprise-user/update` | PUT | 更新成员角色 |
| `/api/v1/member/enterprise-user/delete` | DELETE | 移除成员 |
| `/api/v1/member/enterprise-user/page` | GET | 成员列表（分页） |
| `/api/v1/member/enterprise-user/get-me` | GET | 获取当前用户角色 |
